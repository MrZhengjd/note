三个主要命令
 git add .			先添加修改代码
 git commit -m ""		然后提交代码
 git push 			同步到远程代码库
 git pull origin master 	拉取远程最新代码
 git init			把本地目录设为一个仓库
 git remote add origin https://github.com/MrZhengjd/learning.git
 git push https://github.com/MrZhengjd/learning.git master


git 修改远程代码步骤
先在远程创建好git目录，然后再电脑上的目录通过git clone ..来拉取目录内容
然后通过git add .
git commit -m ""
git push
来上传代码
